# 集成与配置设计

## 1. 配置中心目标

平台需要通过配置接入不同项目、不同模型、不同需求来源、不同通知渠道和不同执行环境，避免硬编码。

配置分层：

| 层级 | 说明 |
| --- | --- |
| 系统级配置 | 全局模型、全局通知、用户认证、安全策略 |
| 项目级配置 | 项目模型、需求来源、字段映射、通知策略、设备池 |
| 计划级配置 | 执行环境、定时策略、用例范围、报告策略 |
| 用例级配置 | 执行器、脚本、命令、断言、超时 |

## 2. 大模型配置

### 2.1 支持类型

| 类型 | 示例 |
| --- | --- |
| 本地模型 | Ollama、LM Studio、vLLM、Xinference |
| 企业模型网关 | 内部 OpenAI-compatible API |
| 云模型 | DeepSeek、通义千问、智谱 GLM、OpenAI-compatible 服务 |
| Mock 模型 | 本地开发和自动化测试使用 |

### 2.2 OpenAI-compatible 配置示例

```json
{
  "provider": "openai_compatible",
  "name": "local-qwen",
  "base_url": "http://localhost:8000/v1",
  "model_name": "qwen2.5-32b-instruct",
  "api_key_secret_ref": "secret://model/local-qwen",
  "parameters": {
    "temperature": 0.2,
    "top_p": 0.8,
    "max_tokens": 4096
  },
  "timeout_seconds": 120
}
```

### 2.3 模型场景绑定

| 场景 | 推荐模型参数 |
| --- | --- |
| 需求摘要 | temperature 0.2，输出稳定 |
| 用例生成 | temperature 0.2-0.4，兼顾覆盖和一致性 |
| 报告解读 | temperature 0.3，允许一定总结能力 |
| 日志归因 | temperature 0.1-0.2，强调证据和准确性 |

## 3. Jira 接入

### 3.1 配置项

```json
{
  "type": "jira",
  "base_url": "https://jira.example.com",
  "auth_type": "pat",
  "project_key": "HW",
  "jql": "project = HW AND issuetype in (Story, Task) ORDER BY updated DESC",
  "field_mapping": {
    "external_id": "key",
    "title": "summary",
    "description": "description",
    "priority": "priority.name",
    "status": "status.name",
    "module": "components[0].name",
    "version": "fixVersions[0].name"
  },
  "sync_interval_minutes": 60
}
```

### 3.2 同步策略

- 支持手动同步和定时同步。
- 以 external_source + external_id 做幂等。
- Jira 更新时间大于本地更新时间时创建需求版本。
- 字段映射失败的记录进入同步异常队列。

## 14. 日志存储、自动采集与环境初始化配置

### 14.1 推荐存储方案

优先推荐 S3-compatible 对象存储，私有化部署使用 MinIO，企业已有云资源可使用对象存储或云盘。

| 存储 | 适用场景 |
| --- | --- |
| MinIO | 内网私有化部署首选 |
| AWS S3/火山 TOS/阿里 OSS/腾讯 COS | 企业已有云对象存储 |
| 飞书云文档/云盘 | 希望报告和日志在飞书内协作查看 |
| 企业 NAS | 实验室大文件、长期归档 |
| 本地文件系统 | 单机开发环境 |

### 14.2 MinIO/S3 配置

```json
{
  "storage": {
    "provider": "minio",
    "endpoint": "http://minio.example.local:9000",
    "bucket": "autotesthub-artifacts",
    "region": "us-east-1",

    "secret_key_secret_ref": "secret://storage/minio/secret_key",
    "signed_url_ttl_seconds": 604800,

  }
}
```
